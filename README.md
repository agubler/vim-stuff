
install vundle
```
git clone git://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```

vim.rc
```
git clone git://github.com/matt-gadd/vim-stuff.git ~/
```

if errors throw, try running
```
:BundleInstall
```

make command-t
```
cd ~/.vim/bundle/command-t/ruby/command-t
ruby extconf.rb
make
```

thats all folks.
